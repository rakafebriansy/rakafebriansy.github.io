Tentang Saya

Kehidupan Pribadi

Saya lahir di Sumenep pada tanggal 29 Februari 2004. Saat ini saya menempuh pendidikan di Universitas Jember program studi Sistem Informasi semester 3 dengan IPK sebesar 3.83.

Saya memiliki hobi bermain video game. Jenis video game yang saya sukai adalah RPG, sebuah genre video game dimana kita dapat membuat karakter kita sesuai dengan yang kita inginkan.

Pemrograman

Saya memiliki ketertarikan terhadap pengembangan perangkat lunak terutama pada pemrograman perangkat lunak. Saya pertama kali mempelajari pemrograman di bangku kuliah dengan bahasa pemrograman Python. Untuk saat ini sedang mempelajari pengembangan aplikasi berbasis web dengan menggunakan HTML 5, CSS 3, Javascript, dan PHP, serta beberapa framework seperti Bootstrap dan Tailwind untuk CSS.

PORTFOLIO

Project Terbaru

Berikut merupakan beberapa aplikasi yang telah saya kembangkan dengan berbagai bahasa pemrograman yang telah saya pelajari.

Si Satset

Si Satset merupakan aplikasi manajemen pengiriman barang antar kota berbasis desktop yang dibuat dengan bahasa pemrograman Python. Aplikasi ini mengimplementasikan beberapa algoritma untuk memecahkan beberapa masalah seperti Pengurutan data dengan Quick sort, Pencarian string dengan Boyer Moore, Knapsack Problem dengan pendekatan Greedy, dan Shortest Path Problem dengan Dijkstra.
